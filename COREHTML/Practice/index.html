HTML Practice Webpage

A webpage made while practicing what I learn by following the "Web Development Mastercourse", published by Tim Corey via IAmTimCorey.com.

HTML topics covered:

  • The structure behind HTML

  • Assigning a title and favicon to a webpage

  • A division of a page used to hold other tags

  • Used to hold a block of text

  • Ordered and unordered list elements

  • Display information in a structured table layout

  • Labels, inputs, buttons, form actions, and methods

  • Semantic HTML structure and element usage

  • The lang attribute for defining document language

  • Help assistive technologies identify elements

  • The scope attribute for improved accessibility

  • The tabindex attribute for keyboard navigation

CSS topics covered:

  • Styling an element using the style attribute, generally avoided due to maintenance difficulty

  • Using an external stylesheet with the <link> tag to apply global styling across elements

  • Applying styles using classes (.class) and IDs (#id) to target specific elements

  • Fixed sizing with px and responsive sizing with em and rem

  • Using font-family and line-height, including importing custom fonts

  • Defining colors using #hex, rgba(), and hsla()

  • Alignment, transformation, indentation, and decoration using text-related CSS properties

  • Showing and hiding elements and understanding layout behavior using display, visibility, and related properties

  • Adding whitespace using margin and padding properties

  • Floating images with float and containing floats using display: flow-root

  • Understanding selector priority and specificity in the CSS cascade

JavaScript Topics Covered:

  • Placing JavaScript in the HTML using the <script> tag, commonly placed before the closing </body> tag for performance

  • Printing information to the console using browser developer tools

  • Moving JavaScript into a separate .js file for readability, reusability, and best practices

  • Linking an external file using the <script src="js/app.js"> tag

  • Declaring values using var, let, and const

  • Performing string interpolation and arithmetic using +, -, /, *, ** operators

  • Adding and removing items using .push() and .pop(), and creating new arrays using .filter()

  • Controlling program flow using if, else if, and else branches, and evaluating values against multiple cases using the switch structure

  • Comparing values with == for loose equality and === for strict equality, and combining conditions using && (and) and || (or)

  • Repeating actions using for loops, for...of iteration, .forEach() callbacks, and while loops

  • Creating reusable code blocks using full body functions and concise expressions using arrow functions

  • Organizing data using key-value pairs, including nested objects and functions stored as object properties

  • Extracting values from objects into variables using concise syntax

  • Convering objects to JSON strings with JSON.stringify() and reading JSON data with JSON.parse()

  • Creating reusable blueprints for objects using the class keyword, initializing data through the constructor, and instantiating new objects using new

  • Defining public fields and private fields using #, and controlling access or formatting through getters and setters

  • Creating functions inside classes using traditional method syntax or arrow functions, with arrow functions preserving this

  • Encapsulating private variables, functions, and classes inside a namespace created by an IIFE to avoid polluting the global scope

  • Controlling where variables can be accessed using global scope, function scope with var, and block scope with let and const

  • Enabling Strict Mode by adding the 'use strict' directive to prevent unsafe behaviors such as undeclared variables, accidental globals, and silent errors

  • Selecting elements using document.getElementById() for #id values, or using querySelector() and querySelectorAll() for any CSS-style selector

  • Changing content, removing elements, applying inline styles, and managing classes using element properties and classList methods

  • Navigating between related elements by moving through children, parents, or siblings from an existing reference instead of running new global queries

  • Generating and inserting elements dynamically using innerHTML, createElement(), and DocumentFragment with attention to performance and safety tradeoffs

  • Responding to user interactions by listening for events with addEventListener(), controlling default behavior and propagation, and updating the DOM through event handlers

Deploying Websites:

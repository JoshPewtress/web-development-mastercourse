A HTML Practice Webpage

A webpage made while practicing what I learn by following the "Web Development Mastercourse", published by Tim Corey via IAmTimCorey.com.

HTML topics covered:

  • Fundamental HTML Tags - The structure behind HTML.
  • Page Head Tags - Assigning a Title and Favicon to a webpage.
  • Header Tags - H1 - H6, and best practices for using them.
  • Div Tag - A division of a page used to hold other tags.
  • Paragraph Tag - Used to hold a block of text.
  • Span Tag - Seperate specific text to assign styling.
  • Lists - Ordered and Unordered.
  • Tables - Display information in a table format.
  • Forms - Labels, Inputs, Buttons, Form Actions and Methods.
  • Accessibility - Semantic structure tags and their uses.
  • Setting page language - lang attribute.
  • Aria Attributes and Roles - Assisting in screen reader element identification.
  • Table Options - Scope attribute.
  • Tab Order - tabindex attribute

CSS topics covered:

  • Inline Style - Applying style through using the style attribute, commonly best to avoid using this method due to difficulty of maintaining and updating.
  • Stylesheet - Use of a cascading stylesheet and the partnering link tag to apply the same style to multiple elements
  • Targetting - The use of Classes and Id's to apply targetted styles at specific elements.
  • Sizing - Fixed sizing with px, and Responsive sizing with em and rem.
  • Fonts - Font-family and line-height attributes, importing fonts.

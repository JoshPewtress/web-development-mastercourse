A HTML Practice Webpage

A webpage made while practicing what I learn by following the "Web Development Mastercourse", published by Tim Corey via IAmTimCorey.com.

Topics covered so far:

  • Fundamental HTML Tags - The structure behind HTML.
  • Page Head Tags - Assigning a Title and Favicon to a webpage.
  • Header Tags - H1 - H6, and best practices for using them.
  • Div Tag - A division of a page used to hold other tags.
  • Paragraph Tag - Used to hold a block of text.
  • Span Tag - Seperate specific text to assign styling.
  • Lists - Ordered and Unordered.
  • Tables - Display information in a table format.
  • Forms - Labels, Inputs, Buttons, Form Actions and Methods.
  • Accessibility - Semantic structure tags and their uses.
